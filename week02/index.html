Chapter 2: Programming Basics

  • Why is it not necessary to add semicolons in JavaScript?

    JavaScript interpreters use Automatic Semicolon Insertion (ASI). But, it is considered best practice two write and terminate each statement on a new line to avoid causing confusion with automated validators and minifiers.
  • Primitive Data Types
    • String
    • Symbol - ES6+
    • Number
    • Boolean
    • Undefined
    • Null
  • String Escape Characters
      Single quote: '`
      Double quote: `"
      End of line: \n
      Carriage return: \r
      Tab: \t

  • What is a symbol?

    A symbol is a primitive value that was introduced in ES6. It can be used to create unique values. They are used as object property keys.

  • Falsy Values

    • ""
    • ''
    • ``
    • 0
    • -0
    • NaN
    • false
    • null
    • undefined

Chapter 3: Arrays, Logic, and Loops

  • Array Methods
    • pop - Remove the last item.
    • push - Append an item to the end.
    • shift - Remove the first item.
    • unshift - Insert an item at the start.
    • concat - Merge one or more arrays.
    • join - Convert all entries into a single comma-delimited string.
    • slice - Create a subarray.
    • splice - Remove an item and insert new items in its place. Returns an array of removed items.
    • reverse - Reverse the order.
    • sort - Sorts items alphabetically.
  • Sets To create a set of words, use the .add() method: new Set().add('the').add('quick'). Here is a list of methods:
    • size - Return the number of values.
    • has - Returns true if an entry exists.
    • delete - Remove a value from the set.
    • clear - Remove all values from the set.
  • Maps Equivalent to 'hash tables' or 'dictionaries' in other languages. Here is a list of methods:
    • set - Add a key/value pair.
    • get - Get a value by its key.
    • has - Check if a key exists.
    • delete - Delete en entry by its key.
    • clear - Remove all key/value pairs.

Chapter 4: Functions

  • Declaration: function hello() { }
  • Expression: const hello = function() {}
  • Functions return undefined by default.

  • What is the difference between parameters and arguments?

    Parameters are set when a function is defined. Arguments are passed to a function when it is invoked.

  • Callback - A function that is passed as an argument to another.
  • Array Iterators - Functions that process elements in an array. They can be chained together.
    • forEach - Iterate over the entries of an array, passing in each value as an argument.
    • map - Convert each entry in an array to a new value.
    • reduce - Accumulate values, two at a time, until a single value remains.
    • filter - Return a new array of items that return true in their callback.
