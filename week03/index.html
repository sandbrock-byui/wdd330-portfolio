Chapter 5: Objects

  • How can I check if a property or method exists on a JavaScript object?

    • Use the in keyword: 'city' in superman
    • Use undefined: superman.city +== undefined
    • Use hasOwnProperty: superman.hasOwnProperty('city')

  • How can I loop through the properties of a JavaScript object?

    • for (const key in superman) { console.log(key + ": " + superman[key]); }
    • for (const value of Object.values(superman)) { console.log(value); }
    • for (const [key, value] of Object.entries(superman)) { console.log(`${key}: ${value}`); }

  • How can I remove properties from a JavaScript object?

    delete superman.fly

  • How can I convert a JSON string to a JavaScript object?

    JSON.parse(batman)

  • How can I convert a JavaScript object to a JSON string?

    • JSON.stringify(myObject)
    • JSON.stringify(wonderWoman, null, " "); // Add newline between key/value pairs

Chapter 6: The Document Object Model

  • Chapter Exercise - heroes.html

  • What is the DOM?

    It is a representation of the HTML document as a network of connected nodes.

    Node types

    • 1 - element
    • 2 - attribute
    • 3 - text
    • 8 - comment
    • 9 - body

    Properties and methods for nodes

    • nodeType - Return an integer containing its type.
    • nodeName - Return the name of an element.
    • Document.body - Return the body element.
    • Document.images - Return a node list of all the images.
    • Document.links - Return a node list of all the a and area elements that have an href attribute.
    • Document.anchors - Return a node list of all the a elements that have a name attribute.
    • Document.forms - Return a node list of all the forms.
    • getElementById() - Return a reference to the element with a unique id attribute.
    • getElementsByTagName() - Return a live node list of elements with the tag name that is provided.
    • getElementsByClassName() - Return a live node list of elements that have the provided class name.
    • querySelector() - Return the first element in a document that matches the CSS selector.
    • querySelectorAll() - Return a node list of all elements in a document that match the CSS selector.
    • childNodes - Return a list of all the child nodes of the selected node.
    • children - Return a list of the child nodes of the selected node that are elements.
    • firstChild - Return the first child of a node.
    • lastChild - Returns the last child of a node.
    • parentNode - Return the parent node of an element.
    • nextSibling - Return the next adjacent node of the same parent.
    • previousSibling - Return the previous adjacent node.
    • nodeValue - Return the value of a text node.
    • textContent - Return the text content of an element as a string.
    • getAttribute() - Return the value of the attribute provides.
    • setAttribute() - Change the value of an element's attribute.
    • className - Get/set the classes of an element directly.
    • classList - Add/remove individual classes to/from an element. classList.add(), classList.remove(), classList.contains()
    • createElement() - Create a new element.
    • createTextNode() - Create a new text node.
    • appendChild() - Add another node as a child node.
    • insertBefore() - Insert a child element at a specific position.
    • removeChild() - Remove a child element.
    • replaceChild() - Replace a child element with another.
    • innerHTML - Return all the child elements as a string of HTML.
    • style - Set camelCase style properties.
    • getComputedStyle() - Return all the style information of an element.

Chapter 7: Events

  • Chapter Exercise - events.html

  • How do I connect event handlers?

    • Inline (not ideal) - <p onclick="console.log('You Clicked Me!')">Click Me</p>
    • Older (better) - document.onclick = function (){ console.log('You clicked on the page!'); }
    • Event Listeners (best) - document.body.addEventListener('click',doSomething);

  • Types of Events

    • click - Occurs when a mouse button is clicked.
    • mousedown - Occurs when the mouse button is pressed down.
    • mouseup - Occurs when the mouse button is released.
    • dblclick - Occurs when the mouse buttin is clicked twice.
    • mouseover - Occurs when the pointer is placed over an element.
    • mouseout - Occurs when the mouse moves away from an element.
    • mousemove - Occurs when the mouse moves over the top of an element.
    • keydown - Occurs when the user pressed down a keyboard key. Includes modifier key information: shiftKey, ctrlKey, altKey, metaKey
    • keypress - Occurs after a keydown but before a keyup event.
    • keyup - Occurs when a key is released.
    • touchstart - Occurs when the user begins to touch the screen.
    • touchend - Occurs when the user stops touching the surface.
    • touchmove - Occurs after the user touches the screen and moves around without leaving.
    • touchenter - Occurs when the user is already touching the screen and passes over an element.
    • touchleave - Occurs when the user is still touching the surface but leaves an element.
    • touchcancel - Occurs when a touch event is interrupted.
