Chatper 8: CSS3 Transforms and Transitions

  • Transforms - Lets you translate, rotate, scale, and/or skew elements.

    • What is a translation in CSS?

      Translation functions allow you to move elements vertically and/or horizontally.

    • What is scaling in CSS?

      Scales an element horizontally then vertically.

    • What is a rotation in CSS?

      It rotates an element around the point of origin by a specified angle value.

    • What is a skew in CSS?

      Skews an element around the X and Y axis.

  • Transitions - Animation by gradual CSS changes

    • transition-property

      Defines the CSS properties of the element that should be transitioned.

    • transition-duration

      Sets how long the transition will take. 200ms is considered the optimum time.

    • transition-timing-function

      Lets you control the pace of the transition in even more granular detail. The most common timing functions include ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier.

    • transition-delay

      Introduce a delay before the transition begins.

  • Animations - Controlled with keyframes. Need -webkit- prefix.
    • keyframes - @keyframes moveRight { from { ... } to { ... } }
    • animation-name - The name of the animation.
    • animation-duration - Defines the length of time (in seconds or milliseconds) to complete on iteration.
    • animation-timing-function - Determines how the animation will progress over its duration.
    • animation-iteration-count - Lets you define how many times the animation will play.
    • animation-direction - Change the direction the animation moves.
    • animation-delay - Define how many milliseconds or seconds to wait before starting the animation.
    • animation-fill-mode - Defines what happens before the first animation iteration begins and after the last animation iteration concludes.

Chapter 12: Canvas, SVG, and Drag and Drop

  • Canvas

    • What is the default canvas width and height?

      The default width is 300 and the defualt height is 150.

    • How do I draw on a canvas?

      You must draw using JavaScript.

    • How do I initialize canvas?

      var canvas = document.getElementById("myCanvas");

      var context = canvas.getContext("2d");

    • Drawing a rectangle with a red border and a blue fill.

      context.strokeStyle = "red";

      context.fillStyle = "blue";

      context.fillRect(10, 10, 100, 100);

      context.strokeRect(10, 10, 100, 100);

    • Drawing an image repeatedly.

      var img = new Image();

      img.src = "../images/bg-bike.png";

      img.onload = function() {

      var pattern = context.createPattern(img, "repeat");

      context.fillStyle = pattern;

      context.fillRect(10, 10, 100, 100);

      context.strokeRect(10, 10, 100, 100);

      };

      }

    • How do I create a gradient?

      Use the createLinearGradient() or createRadialGradient() methods.

    • How do I draw a circle?

      ...

      context.beginPath();

      context.arc(50, 50, 30, 0, Math.PI*2, true);

      context.closePath();

      context.strokeStyle = "red";

      context.fillStyle = "blue";

      context.lineWidth = 3;

      context.fill();

      context.stroke();

    • Save canvas to an image

      window.open(canvas.toDataURL("image/png")

    • Draw image to canvas

      var canvas = document.getElementById("demo6");

      var context = canvas.getContext("2d");

      var image = document.getElementById("myImageElem");

      context.drawImage(image, 0, 0);
