  • Reading Exercise: Master Mobile UX

    • How to Rewire Your Brain for Mobile First UX
      • Desktop to mobile is like trying to shove a marshmallow into a piggy bank.
      • Desktop sales have been on the decline since 2011.
    • How to Improve UX by Understanding Users
      • Be There - It will deepen their loyalty.
      • Be Useful - Immediate. Relevant. Frictionless.
      • Be Quick - People don't have attention spans anymore.
        • Reduce image usage throughout sites/apps
        • SVG/Webfonts are lower bandwidth
        • Optimize and minify CSS and JS files
        • GZIP files to eliminate unnecessary data
      • Reachability Matters - 75% of people rely on thumbs. 49% use a one-handed grip.
    • Better Mobile UX with Strategic Menu Design
      • Navigation - Move navbar to bottom of screen
      • Perception Matters - "Menu" in a square had 20% more clicks
    • Design Intuitive and Mobile Friendly Forms
      • Don't use labels
      • Increase input height
      • Use autofill on forms
    • Designing a Better Mobile Advertising User Experience
      • "The popup is one of the most detested things on the web..."
      • Make ads small and simple
