Week 9

Chapter 9: The Window Object

  • What is the BOM - Browser Object Model?

    A collection of properties and methods that contain information about the browser and computer screen.

  • Dialogs

    • alert() - Shows a message and returns undefined.
    • confirm() - Shows a message with OK/Cancel and returns true or false.
    • prompt() - Shows an input box and returns the string value or null.

  • How do I tell what browser the application is runnin gin?

    window.navigator.userAgent

  • window.location properties and methods
    • window.location.href - Gets the URL of the current page
    • window.location.protocol - Gets the protocol of the current page (http, https, pop2, ftp)
    • window.location.hostname - Gets the hostname of the current page (www.sitepoint.com)
    • window.location.port - Gets the port of the current page (This will be blank if not specified in the URL)
    • window.location.pathname - Gets the path of the current page (/premium/books/javascript-novice-to-ninja)
    • window.location.search - Gets the query string (?q=javascript&limit=24)
    • window.location.hash - Gets the fragment identifier of the current page (#test)
    • window.location.origin - Gets the origin of the current page (protocol + domain)
    • window.location.reload() - Reload the page. If true is passed in as a parameter, it will force reload from the server.
    • window.location.assign() - Load another resource from a URL.
    • window.location.replace() - Similar to assign() but it does not store the current page in the session history.
    • window.location.toString() - Returns a string containing the whole URL.
  • window.history properties and methods
    • window.history.length - Shows how many pages have been visited before arriving at the current page.
    • window.history.go() - Method can be used to go to a specific page. 1 = forward, 0 = current, -1 = back
    • window.history.forward() - Navigates forward.
    • window.history.back() - Navigates back.
  • window properties and methods
    • window.open() - Opens a new window to the specified URL.
    • windowref.close() - Closes the referenced window.
    • window.moveTo(0,0) - Moves the window to the top-left corner.
    • window.resizeTo(600,400) - Resizes the window to the specified width and height.
  • window.screen properties and methods
    • window.screen.height - Returns the screen height in pixels.
    • window.screen.width - Returns the screen width in pixels.
    • window.screen.availHeight - Returns the screen height, excluding any menus.
    • window.screen.availWidth - Returns the screen width, excluding any menus.
    • window.screen.colorDepth - Returns the color bit depth of the user's monitor.
  • document.write('') - Writes a string of text to the page.
  • cookies
    • How do I set a cookie? - document.cookie = 'hero=true'
    • How do I read a cookie? - document.cookie.split("; ")
    • How do I set a cookie's expiration? - `expires=${ expiryDate.toUTCString() }` or 'name=Batman; max-age=86400'
    • How do I set the path of a cookie? - 'path=/'
    • How do I set the domain of a cookie? - 'domain=sitepoint.com'
    • How do I ensure a cookie will only be sent over HTTPS? - 'secure'
    • How do I delete a cookie? Set it to expire at a time in the past. Session cookies will expire when a tab or window is closed.
  • timing functions
    • window.setTimeout(func, int) - Accepts a callback to a function as its first parameter and a number of milliseconds as its second.
    • window.clearTimeout(int) - Cancels the timeout that returned the integer parameter.
    • window.setInterval(func, int) - Repeatedly calls the callback after every given number of milliseconds.
    • window.clearInterval(int) - Cancels the interval that reutrned the integer parameter.
  • window.requestAnimationFrame - Similar to window.setInterval except it uses better graphics-handling and it doesn't run when the tab is inactive.

Chapter 14: HTML5 APIs

  • What is the data- attribute used for?

    It is used for embedding custom attributes that are ignored by the browser.

  • How do I get the value of a data- attribute?

    Either element.dataset.name or element.getAttribute('data-name')

  • web storage - window.localStorage

    • How do I save data to localStorage?

      localStorage.setItem('name', 'value'); or localStorage.name = 'value';

    • How do I load data from localStorage?

      const val = localStorage.getItem('name'); or const val = localStorage.name;

    • How do I remove a value from localStorage?

      localStorage.removeItem('name'); or delete localStorage.name;

    • How do I remove everything from localStorage?

      localStorage.clear();

  • geolocation

    • How do I get the user's location?

      navigator.geolocation.getCurrentPosition(callbackFunc); Reference coords.latitude and coords.longitude.

    • position.speed - Returns the ground speed of the device in meters per second.
    • position.altitude - Returns an estimate of the device's altitude.
    • position.heading - Returns the direction the device is moving in.
    • position.timestamp - Returns the time that the position information was recorded.
    • navigator.geolocation.watchPosition(func) - Calls func every time the position is updated.
    • navigator.geolocation.clearWatch(id) - Clears a watchPosition callback.
  • web workers

    • How do I create a web worker?

      const worker = new Worker('task.js');

    • How do I send data to/from web workers?

      worker.postMessage('data') and from within the worker use self.postMessage('data')

    • How do I receive data from a web worker?

      worker.addEventListener('message', (event) => {...}, false);

    • How do I stop a web worker?

      worker.terminate(); or from within the worker use self.close();

  • What are service workers?

    They allow a worker script to run in the background with the added benefit of being able to intercept network requests.

  • What is Websockets?

    It is a new protocol that allows two-way communication.

  • Notifications

    • How do I ask the user for permission to display notifications?

      Notification.requestPermission()

    • How do I check if the user has granted permission to show notifications?

      if (Notification.permission === 'granted') { ... }

    • How do I show a notification?

      new Notification('Hello JavaScript!', { body: 'hi', icon: 'image.png' });

  • audio/video properties and functions
    • video.play() - Plays a video
    • video.pause() - Pauses a video
    • video.volume - Sets or gets the volume for a video
    • video.muted - Sets or gets whether or not a video is muted.
    • video.currentTime - Sets or gets the current position.
    • video.playbackRate - Fast-forward at 8x speed
    • video.loop - Set true to repeat in a loop
    • video.duration - - How long the video lasts
